## Install ESP-IDF Extension in VS Code

1.  Open Visual Studio Code
2.  Install the ESP-IDF extension:
    
    -   Go to the Extensions view (Ctrl+Shift+X)
    -   Search for "ESP-IDF"
    -   Click "Install" on the official Espressif ESP-IDF extension
    
3.  Configure the ESP-IDF extension:
    
    -   Press Ctrl+Shift+P to open the command palette
    -   Type "ESP-IDF: Configure ESP-IDF extension" and select it
    -   Choose "Express"
    -   Select the same ESP-IDF version you installed standalone
    -   Set the ESP-IDF directory to where you installed it (e.g. C:\\esp\\esp-idf)
    -   Set the ESP-IDF tools directory (usually C:\\Espressif)
    -   Click "Install" and wait for the process to complete

#### Steps to use a previous version of ESP-IDF:

1. Determine which version of ESP-IDF you want to use. You can check the available releases on the ESP-IDF GitHub repository: [https://github.com/espressif/esp-idf/releases](https://github.com/espressif/esp-idf/releases)
2. Clone the specific version of ESP-IDF you want to use. For example, to use v4.3.3:

bash

`git clone -b v4.3.3 --recursive https://github.com/espressif/esp-idf.git esp-idf-v4.3.3`

3. Change to the new ESP-IDF directory:

bash

`cd esp-idf-v4.3.3`

4. Run the installation script:

bash

`./install.sh`

5. Set up the environment variables for this version:

bash

`. ./export.sh`

6. Now you can use this specific version of ESP-IDF for your project.
